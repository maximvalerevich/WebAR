# ArtVision - Virtual Art Try-On

ArtVision — это современное веб-приложение для виртуальной «примерки» картин в интерьере. Оно позволяет пользователям загружать фотографии своих стен и реалистично размещать на них произведения искусства с учетом масштаба, перспективы и освещения.

## 🚀 Основные возможности

- **Виртуальный холст (Canvas):** Точное размещение картин на фото стены с помощью Konva.js.
- **Инструмент «Линейка» (Ruler):** Калибровка реального масштаба для точного отображения размеров картины.
- **Коррекция перспективы (Corner Pin):** Искажение изображения для соответствия углу съемки стены.
- **Настройка теней:** Динамические тени для создания эффекта глубины и реализма.
- **AR-просмотр:** Просмотр картин в дополненной реальности (WebXR) с помощью Google Model-Viewer.
- **Экспорт:** Сохранение результата в форматах JPG/PNG.

## 🛠 Технологический стек

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Canvas Logic:** React-Konva
- **Validation:** Zod
- **AR/3D:** Google Model-Viewer

## 📦 Начало работы

### Требования

- Node.js 18.x или выше
- npm / yarn / pnpm

### Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/maximvalerevich/web-ar.git
   cd web-ar
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```

4. Откройте [http://localhost:3000](http://localhost:3000) в браузере.
